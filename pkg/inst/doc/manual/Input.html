Input files for the IsoGeneGUI
The input are information about dose and gene expression which can be excel files, text files, or from objects in an R workspace file (*.RData). The format of the response/gene expression data can be a vector or it can be a matrix or table where the columns are the arrays and the rows are the gene names or IDs. The dose information should be a vector or table contain the dose which corresponds to the arrays in the expression matrix/table.
