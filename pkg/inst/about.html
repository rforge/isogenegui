About the IsoGeneGUI
The IsoGene Graphical User Interface (IsoGeneGUI) is a user friendly interface of the IsoGene package which is aimed to perform analysis of dose-response studies in microarray experiments. The IsoGeneGUI is developed for the user with no or limited knowledge about R programming so he/she can perform the analysis of dose-response in microarray setting  easily. This GUI was developed using tcl/tk.

Screenshot of the main IsoGeneGUI window
The GUI has four main menus and in the middle of the window there is an info box. The package has four main menus: File, Analysis, Plots and Help.

File:
      1. Open Data
          a. R workspace (*.RData files)
          b. Excel or text file (*.xls or *.txt files)
      2. Show Data
      3.  Exit
Analysis: 
     1. Set seed
     2. Likelihood Ratio Test E2 Analysis
     3. Permutation Analysis
     4. Significant Analysis of Microarrays (SAM)
         a. SAM Permutation
         b. SAM Analysis

Plot:       
    1. IsoPlot
    2.Permutation Plot
    3.SAM Plot
        a. Plot of FDR vs. Delta
        b. Plot of number of significant genes vs. Delta
        c. Plot of number of False Positive vs. Delta
    4. User defined scatter plot

Help:   
  1. IsoGene Help
  2.  IsoGeneGUI Help
  3. About
